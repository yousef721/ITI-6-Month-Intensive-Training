1 Events Exercise - Company & Employee System

Scenario: Design a company management system where salary increases trigger budget updates through events.
  • Use the previous designed class Employee (at least has Name & Salary)
  • Design a new class Company. Each company has a Name, Budget and a list of working employees.
  • Add function IncreaseSalary(float amount) to class Employee that increases employee salary by the given amount. When this happens, the company must be notified by that increase so it decreases this amount from its budget.
Key Concept: Use events to notify the Company when an Employee's salary increases.

2 Delegates Exercise - Employee Filtering

Task: In class Company, write a function that filters the employee list using a delegate.
Function Signature:
List<Employee> FilterEmployees(filter)

where filter is a delegate of your choice
Key Concept: Use delegates to create flexible filtering logic for the employee list.
